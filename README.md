## **🏗 Project Structure**

### **Student.java**

Handles student information and seat assignments.

### **Course.java**

Handles student lists and course management.

### **AttendanceApp.java**

Runs the main program and handles user interaction.

## **🛡 Input Validation Rules**

- Seat numbers must be valid
- Course name cannot be blank
- Students cannot have duplicate seats
